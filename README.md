💬 ConnectMe - Real Time Chat App
🚀 About The Project

ConnectMe is a full-stack real-time chat application where users can chat instantly, share images and videos, see who is online, and customize the app with different themes and wallpapers.

I built this project using React, Node.js, Express.js, MongoDB, Socket.io, Clerk Authentication, and ImageKit.

This project helped me learn how real-world chat applications work and how to build and deploy a complete full-stack application.

 🚀 Live Demo - https://connectme-gfuq.onrender.com/auth
